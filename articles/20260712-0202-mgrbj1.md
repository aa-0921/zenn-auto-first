## コールバック地獄からの脱出：Promise の登場

Promise は、非同期処理の結果をオブジェクトとして扱うことができるようにするための仕組みです。Promise を使用することで、コールバック地獄から脱出でき、より可読性の高いコードを書くことができます。

例えば、以下のようなコードはコールバック地獄の典型的な例です:

```javascript
getData(function(data) {
  doSomething(data, function(result) {
    doSomethingElse(result, function(finalResult) {
      console.log(finalResult);
    });
  });
});
```

このコードは、ネストしたコールバック関数が複雑になり、可読性が低下します。これに対して、Promise を使用したコードは以下のようになります:

```javascript
getData()
  .then(data => doSomething(data))
  .then(result => doSomethingElse(result))
  .then(finalResult => console.log(finalResult));
```

このコードは、Promise の then メソッドを使用することで、コールバック地獄から脱出でき、より可読性の高いコードを書くことができます。

## then と async/await の使い分け

Promise の then メソッドは、非同期処理の結果をオブジェクトとして扱うことができるようにするための仕組みです。しかし、Promise の then メソッドは、コールバック関数を使用するため、依然としてコールバック地獄の可能性があります。

これに対して、async/await は、Promise を使用することで、非同期処理を同期的に書くことができるようにするための仕組みです。async/await を使用することで、コードがよりシンプルになり、可読性が向上します。

例えば、以下のようなコードは Promise の then メソッドを使用したコードです:

```javascript
getData()
  .then(data => doSomething(data))
  .then(result => doSomethingElse(result))
  .then(finalResult => console.log(finalResult));
```

このコードは、Promise の then メソッドを使用することで、コールバック地獄から脱出できますが、依然としてコールバック関数が使用されています。

これに対して、以下のようなコードは async/await を使用したコードです:

```javascript
async function fetchData() {
  const data = await getData();
  const result = await doSomething(data);
  const finalResult = await doSomethingElse(result);
  console.log(finalResult);
}
```

このコードは、async/await を使用することで、非同期処理を同期的に書くことができ、コードがよりシンプルになり、可読性が向上します。

## エラーハンドリングの基本

Promise と async/await を使用する場合、エラーハンドリングも重要です。Promise では、catch メソッドを使用することで、エラーを扱うことができます。一方、async/await を使用する場合は、try/catch を使用することで、エラーを扱うことができます。

例えば、以下のようなコードは Promise の catch メソッドを使用したコードです:

```javascript
getData()
  .then(data => doSomething(data))
  .catch(error => console.error(error));
```

このコードは、Promise の catch メソッドを使用することで、エラーを扱うことができます。

これに対して、以下のようなコードは async/await を使用したコードです:

```javascript
async function fetchData() {
  try {
    const data = await getData();
    const result = await doSomething(data);
    console.log(result);
  } catch (error) {
    console.error(error);
  }
}
```

このコードは、async/await を使用することで、非同期処理を同期的に書くことができ、エラーを扱うことができます。
