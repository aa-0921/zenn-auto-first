## なぜ30代からエンジニアを目指したのか

### きっかけは「自動化の楽しさ」だった

建築現場での作業効率化を試みていたときに、Excelマクロや簡単なスクリプトで業務を自動化できた時の喜びがきっかけでした。「もっとシステムとして作れたら」と思い、プログラミングスクールに通うことにしました。

### 年齢を理由にする前に「まずはやってみる」

周りからは「もう遅いんじゃない？」といった意見もありましたが、実際に手を動かしてみると、年齢が武器になることが多いことに気づきました。

## 学習の進め方：3つのフェーズで実践した

### フェーズ1: 基礎固め（3ヶ月間）

まずはHTML/CSS/JavaScriptの基礎を固めました。以下のような学習スケジュールをつけました：

```javascript
// 毎日30分、基本構文を書く習慣
function greet(name) {
  return `こんにちは、${name}さん`;
}
console.log(greet("田中"));
```

- 平日: 平日の夜に2時間学習
- 休日: 8時間集中学習
- 毎日コードを書く習慣をつける

### フェーズ2: 実践プロジェクト（4ヶ月間）

簡単なTODOアプリを作った後、自分の趣味である旅行記録アプリを開発しました。

```javascript
// TODOアプリの簡易版
const todos = [];

function addTodo(text) {
  todos.push({
    id: Date.now(),
    text: text,
    completed: false
  });
}
```

### フェーズ3: ポートフォリオ作成（2ヶ月間）

GitHub Pagesでポートフォリオサイトを公開し、転職活動を開始しました。

## 現場での手応え：年齢の強みとして活かした

### コミュニケーション能力の差別化

建築業界での経験が活かり、チーム内での調整やクライアントとのコミュニケーションがスムーズに進みました。特に以下の場面で実力を発揮しました：

- チーム内の技術的な問題解決
- プロジェクト進行上の課題提起
- クライアントへの要件ヒアリング

### 深い理解力での問題解決

年齢のためか、他の新人よりもしっかりと基本を理解している方が多いと感じました。例えば：

```javascript
// 他の新人が書いたコード
let data = fetch('/api/data').then(res => res.json());

// 自分が書いたコード（エラーハンドリングも含めて）
async function fetchData() {
  try {
    const response = await fetch('/api/data');
    if (!response.ok) throw new Error('Network response was not ok');
    const data = await response.json();
    return data;
  } catch (error) {
    console.error('Fetch error:', error);
    return null;
  }
}
```

## 後から振り返って良かった3つの決断

### 1. 独学での学習継続

プログラミングスクールは利用しましたが、基本的な学習は独学で進めました。公式ドキュメントやMDNを読む習慣が後々重宝しました。

### 2. 現場のエンジニアとネットワーク構築

QiitaやTwitterで現場のエンジニアとつながり、業務中の疑問を気軽に相談できる環境を作りました。

### 3. 完璍型の考え方を捨てる

完璧を目指すあまりに挫折しないように、「まずは動くものを作る」ことを重視しました。

## 同じ道を選ぶあなたへ

### 学習時間の確保術

- 朝5時に起きて1時間学習
- 仕事帰りにカフェで2時間
- 週末は8時間集中

### 心構え

- やり始めた時点で既に負けていない
- 年齢は経験の幅を広げる武器
- 完璧よりも継続が重要

### 転職活動のポイント

- ポートフォリオサイトを必ず作成
- 自分のストーリーを明確に伝える
- 年齢は隠さず、経験としてアピール

## まとめ

30代からエンジニアを目指した今でも、時には不安を感じることがあります。しかし、実際に現場で活動してみると、年齢や前職経験がむしろ評価されました。今後エンジニアを目指す人の参考になれば幸いです。

特に重要なのは、「まずはやってみる」という行動することです。完璧を求めず、継続して学び続けることが、きっとあなたの道を切り開いてくれるでしょう。
