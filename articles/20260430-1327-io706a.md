## REST APIとは？
REST（Representational State Transfer）APIは、ウェブサービスとクライアントがデータをやり取りするための通信プロトコルです。フロントエンド開発者にとって、アプリケーションとバックエンドを連携させる重要なツールです。HTMLやJavaScriptを使ったウェブアプリケーションでは、ブラウザからサーバーへのリクエストを送信し、受信したデータを処理する流れが中心になります。

## HTTPメソッドとステータスコードの理解
### REST APIで使うHTTPメソッド
HTTPリクエストの種類は以下の5つのメソッドに分類されます：
- **GET**：リソースの取得（例: ユーザーリストの取得）
- **POST**：リソースの作成（例: 新規ユーザー登録）
- **PUT**：リソースの更新（例: ユーザー情報の変更）
- **DELETE**：リソースの削除（例: アカウントの削除）

それぞれのメソッドは、サーバーがどのようなアクションを実行すべきかを明確に示します。例として、GETリクエストのURLは `/users`、POSTリクエストのURLは `/users` になります。

### ステータスコードの意味
サーバーからの応答には、数字のステータスコードが含まれます。主なものを以下にまとめます：
- **200 OK**：リクエスト成功
- **201 Created**：新規リソース作成成功（POSTレスポンス）
- **400 Bad Request**：リクエスト形式エラー
- **404 Not Found**：リソースが見つからない
- **500 Internal Server Error**：サーバー内部エラー

これらのコードは、APIの挙動を理解する上で非常に重要です。開発時にはエラーハンドリングでこれらを適切に対応させる必要があります。

## JSONのやり取り：データの構造と送受信
### JSONとは？
JSON（JavaScript Object Notation）は、人間が読みやすい形式でデータを表現する標準形式です。リクエスト・レスポンスで使われるデータ形式として、JavaScriptのオブジェクトや配列に似た構文を使います。

#### 送信例（POSTリクエストのJSONボディ）
```json
{
  "name": "太郎",
  "age": 30,
  "email": "tarou@example.com"
}
```

#### 受信例（GETリクエストのレスポンス）
```json
[
  {
    "id": 1,
    "name": "太郎",
    "email": "tarou@example.com"
  },
  {
    "id": 2,
    "name": "花子",
    "email": "hanako@example.com"
  }
]
```

### JSONの扱い方
JavaScriptでは、`JSON.parse()`で受信した文字列をオブジェクトに変換し、`JSON.stringify()`でオブジェクトを送信用の文字列に変換します。例：
```javascript
// 受信時
const data = JSON.parse(response.body);

// 送信時
const requestBody = JSON.stringify({
  name: "新規ユーザー",
  age: 25
});
```

## fetchとaxiosでの呼び出し例
### fetch APIの使い方（ブラウザ内蔵）
`fetch`はブラウザに組み込まれたAPIです。シンプルにリクエストを送信できます。

#### GETリクエストの例
```javascript
fetch('https://api.example.com/users')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('エラー:', error));
```

#### POSTリクエストの例
```javascript
fetch('https://api.example.com/users', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: '新規ユーザー',
    age: 25
  })
})
  .then(response => response.json())
  .then(data => console.log('登録成功:', data))
  .catch(error => console.error('エラー:', error));
```

### axiosの使い方（ライブラリ）
`axios`はnpmでインストール可能なライブラリで、エラーハンドリングが直感的です。

#### GETリクエストの例
```javascript
axios.get('https://api.example.com/users')
  .then(response => console.log(response.data))
  .catch(error => console.error('エラー:', error));
```

#### POSTリクエストの例
```javascript
axios.post('https://api.example.com/users', {
  name: '新規ユーザー',
  age: 25
})
  .then(response => console.log('登録成功:', response.data))
  .catch(error => console.error('エラー:', error));
```

## まとめ
REST APIをフロントエンドから叩く流れは、HTTPメソッドの選択 → JSON形式のデータを送受信 → fetchやaxiosでリクエストを実装することです。初心者でも、メソッドの意味やステータスコードの理解から始め、実際のコード例を踏まえて実装することで、徐々に慣れていきます。特に、エラーハンドリングやリクエスト・レスポンスの構造を意識することが重要です。実際のプロジェクトでは、ドキュメントを見ながらAPIの仕様を確認し、試行錯誤を重ねるのが基本です。
